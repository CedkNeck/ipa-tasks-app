# Social Recipe

Application sociale de partage de recettes construite avec Nuxt 3, Supabase et Capacitor.

## Configuration requise

- Node.js 16+
- npm ou yarn
- IDE recommandé : Visual Studio Code

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm run test
npm run test:e2e
```
